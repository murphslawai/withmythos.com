# AGENTS.md — Mythos (withmythos.com)

## Project
Coming-soon landing page for Mythos, a creative studio at the intersection of ancient storytelling and modern systems. "Myth + OS." Owner: Josh Walker. Domain: withmythos.com.

## Stack
Single self-contained HTML file. No framework, no build tools. Vanilla HTML/CSS/JS with CSS custom properties. Google Fonts CDN (Cormorant Garamond + Inter). Intersection Observer for scroll animations.

## Commands
No build step. Serve locally with any static server or deploy to Cloudflare Pages.

## Source Map
- `index.html` — the entire site
- `../assets/mythos-logo.svg` — logo wordmark (inlined in HTML)
- `STYLE-GUIDE.md` — full design reference

## Design
- Warm cream background (#F2EFE8), ink-dark text (#2C2A28)
- Cormorant Garamond 300 for headings. Alternating normal + italic lighter-ink sub-lines.
- Blur-to-sharp reveal animations on scroll (1.6s cubic-bezier)
- Noise texture overlay + subtle vignette

## Copy Voice
Jack Butcher style. Short declarative lines. Line breaks do the structural work. No hedging, no softeners.

## What's Not Built Yet
Email capture backend, OG image, favicon, analytics. See `STYLE-GUIDE.md` for full reference.
